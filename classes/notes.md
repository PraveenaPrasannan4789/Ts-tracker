# Classes in TypeScript 

## What is a class?
A class is a blueprint to create objects with properties and methods.

---

## Key Concepts

### 1. Constructor
- Special method used to initialize object values

### 2. Properties & Methods
- Properties → variables inside class
- Methods → functions inside class

---

## Access Modifiers

### public
- Accessible everywhere (default)

### private
- Accessible only inside the class

### protected
- Accessible inside class and subclasses

---

## readonly
- Property cannot be changed after initialization

---

## Inheritance
- One class can extend another using `extends`

---

## Encapsulation
- Hiding internal details using private/protected

---

## Why classes?
- Organize code
- Reusability
- Real-world modeling (User, Product, etc.)

---
